# Launch brief: Custom evaluator builder

## Launch name
Custom evaluator builder

## Launch date
TBD

## What is changing
Teams can now build custom evaluators directly in the platform and run them against existing trace datasets.

## Who this is for
AI engineers who have existing trace data and want to evaluate it against criteria that go beyond built-in evaluators.

## Why it matters
Most evaluation problems are domain-specific. Generic evaluators do not capture the nuances of what good looks like for a specific agent or application.

## Product details
- Build custom evaluators using natural language criteria or code
- Run evaluators against any trace dataset
- Compare results across runs
- Set thresholds and get alerted when quality drops

## Technical workflow
1. Select a trace dataset
2. Define custom evaluation criteria (natural language or code)
3. Run the evaluator
4. View results by span, trace, or aggregate
5. Set a quality threshold
6. Get alerted when future runs fall below threshold

## Primary message
Now build and run custom evaluators tailored to your specific quality criteria.
