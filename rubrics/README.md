# Rubrics

Use rubrics to evaluate content quality before publication.

Score from 1 to 5:
- 5 = Excellent
- 4 = Strong (publishable)
- 3 = Acceptable (needs improvement)
- 2 = Weak (significant issues)
- 1 = Poor (do not publish)

A publishable asset should score 4 or higher on all relevant rubrics.

| Rubric | Use for |
|---|---|
| editorial-quality.md | All publishable content |
| technical-accuracy.md | Technical and product content |
| developer-fluency.md | Developer-facing content |
| seo-quality.md | Blog and search-indexed content |
| brand-fit.md | All publishable content |
| distribution-readiness.md | Content ready for distribution |
| analyst-readiness.md | Analyst-facing materials |
| launch-readiness.md | Launch assets |
