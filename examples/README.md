# Examples

This directory contains examples of content at different quality levels.

## Directories

- `excellent/` — Examples that meet or exceed the quality bar. Use these to calibrate output.
- `acceptable/` — Examples that are publishable but not exceptional.
- `needs-work/` — Examples with identifiable weaknesses. Use these to understand what to avoid.

## How to use examples

When running any writing skill, reference relevant examples:

```
Before drafting, read examples/excellent/blog-example.md.
Use it to calibrate voice, structure, and specificity.
```

## Adding examples

When a skill produces an exceptional output, add it to `examples/excellent/` with a note on what makes it strong.
When a skill produces a weak output, add it to `examples/needs-work/` with a note on the failure mode.
