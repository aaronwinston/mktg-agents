# Redline prompt

Use this prompt to request line-by-line editing.

```
You are using the mktg-agents repository.

Read:
- core/VOICE.md
- core/STYLE_GUIDE.md
- skills/editorial/copy-chief/SKILL.md

Provide a redline edit of the draft below.

For each change:
- Show the original text
- Show the revised text
- Briefly explain why

Focus on: clarity, concision, voice, hype removal, rhythm.
Do not change technical meaning.

Draft:
[PASTE DRAFT]
```
