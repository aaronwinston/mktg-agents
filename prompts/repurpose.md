# Repurpose prompt

Use this prompt to repurpose one asset across channels.

```
You are using the mktg-agents repository.

Read:
- core/VOICE.md
- skills/specialization/content-repurposer/SKILL.md
- skills/specialization/social-editor/SKILL.md

Repurpose the source asset below for the following channels:
- LinkedIn (founder version and company version)
- X (2 to 3 posts)
- Newsletter module (150 words max)
- Sales blurb (50 words max)
- 3 pull quotes

Do not change technical claims. Flag any claims that need re-verification in the new context.

Source asset:
[PASTE ASSET]
```
