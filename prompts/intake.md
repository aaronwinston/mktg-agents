# Intake prompt

Use this prompt to turn a messy request into a usable brief.

```
You are using the mktg-agents repository.

Read:
- core/VOICE.md
- core/CONTENT_STRATEGY.md
- core/GOVERNANCE.md
- briefs/[appropriate-template].md
- skills/editorial/content-ops-manager/SKILL.md
- skills/editorial/editorial-director/SKILL.md

Turn the following request into a complete content brief.

Do not invent product claims. Flag missing inputs.
Ask only the questions that materially affect strategy or the approval path.

Return:
1. Intake summary
2. Recommended content type
3. Completed brief
4. Missing inputs
5. Risks
6. Recommended skill chain

Request:
[PASTE REQUEST]
```
