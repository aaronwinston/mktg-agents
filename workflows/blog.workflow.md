# Blog production workflow

## Trigger
New blog brief is approved.

## Steps
1. Confirm brief is complete using `briefs/blog-brief-template.md`.
2. Run `skills/editorial/editorial-director/SKILL.md`.
3. Run `skills/foundation/ai-researcher/SKILL.md`.
4. Run `skills/foundation/dev-copywriter/SKILL.md` to produce first draft.
5. Run `skills/foundation/dev-reviewer/SKILL.md`.
6. Run `skills/specialization/technical-fact-checker/SKILL.md`.
7. Run `skills/specialization/seo-strategist/SKILL.md`.
8. Run `skills/editorial/copy-chief/SKILL.md`.
9. Run `skills/quality/claims-risk-reviewer/SKILL.md`.
10. Run `skills/quality/final-publish-reviewer/SKILL.md`.
11. Run `skills/specialization/social-editor/SKILL.md`.
12. Run `skills/editorial/content-ops-manager/SKILL.md` to finalize publishing plan.

## Output
- Final draft ready to publish
- Metadata complete
- Social variants ready
- Owner and publish date confirmed
