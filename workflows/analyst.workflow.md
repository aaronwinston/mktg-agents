# Analyst workflow

## Trigger
Analyst note, report, or inquiry received.

## Steps
1. Read and classify the analyst signal.
2. Run `skills/specialization/analyst-relations-writer/SKILL.md`.
3. Run `skills/quality/claims-risk-reviewer/SKILL.md`.
4. Review with executive or AR lead.
5. Run `skills/editorial/copy-chief/SKILL.md` on final response.
6. Run `skills/quality/final-publish-reviewer/SKILL.md`.
7. Send or publish the response.

## Output
- Response or briefing ready
- Claims reviewed
- Executive approved
