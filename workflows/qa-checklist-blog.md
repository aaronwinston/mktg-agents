## MAIN CHECKLIST

### 1. Grammar & Language ✏️

- [ ] **No spelling errors**
  - [ ] Spell check passed
  - [ ] Homophones reviewed (their/there/they're, etc.)
  - [ ] Proper nouns capitalized correctly
  - Comments: ____________

- [ ] **No grammar errors**
  - [ ] Subject-verb agreement correct
  - [ ] Verb tenses consistent
  - [ ] Pronoun reference clear
  - Comments: ____________

- [ ] **Sentence structure appropriate**
  - [ ] Sentences <20 words on average
  - [ ] Varied sentence length
  - [ ] No run-on sentences
  - [ ] Punctuation correct and consistent
  - Comments: ____________

### 2. Writing Quality & Clarity 📝

- [ ] **Tone and voice match brand guidelines**
  - [ ] Conversational vs. formal as appropriate
  - [ ] Jargon explained or avoided
  - [ ] Consistent voice throughout
  - Comments: ____________

- [ ] **Content is clear and easy to understand**
  - [ ] Concepts explained simply
  - [ ] Transitions between sections smooth
  - [ ] Readability score >80 (Flesch-Kincaid)
  - [ ] Paragraph length appropriate (3-5 sentences)
  - Comments: ____________

- [ ] **Structure and organization logical**
  - [ ] Hook paragraph captures attention (first 75 words)
  - [ ] Clear thesis/purpose statement
  - [ ] Main sections support thesis
  - [ ] Conclusion summarizes key points
  - [ ] Flow from section to section clear
  - Comments: ____________

### 3. Accuracy & Verification ✓

- [ ] **All facts and statistics verified**
  - [ ] Data sources cited and credible
  - [ ] Numbers double-checked
  - [ ] Quotes are exact (verified against source)
  - [ ] Dates are accurate
  - Comments: ____________

- [ ] **No false or misleading claims**
  - [ ] Opinions marked as opinions
  - [ ] Exaggerations avoided
  - [ ] Comparisons fair and accurate
  - Comments: ____________

- [ ] **External links are current and functional**
  - [ ] All links tested (no 404s)
  - [ ] Links go to credible sources
  - [ ] Anchor text descriptive (not "click here")
  - Comments: ____________

### 4. SEO Optimization 🔍

- [ ] **Meta information optimized**
  - [ ] Title tag: 50-60 characters, includes keyword
  - [ ] Meta description: 155 characters, compelling
  - [ ] Both are unique (not duplicates)
  - Comments: ____________

- [ ] **Keywords strategically placed**
  - [ ] Primary keyword in: title, first 100 words, headers
  - [ ] Keyword density natural (0.5-1.5%)
  - [ ] Related keywords included
  - [ ] Keyword stuffing avoided
  - Comments: ____________

- [ ] **Headers used for structure**
  - [ ] H1 used once (title)
  - [ ] H2/H3 used for section hierarchy
  - [ ] Headers include keywords where natural
  - [ ] Headers descriptive and useful
  - Comments: ____________

### 5. Content Completeness 📋

- [ ] **All required components present**
  - [ ] Title (compelling, keyword-focused)
  - [ ] Featured image with alt text
  - [ ] Hook paragraph (50-75 words)
  - [ ] Table of contents
  - [ ] 4-5 main body sections
  - [ ] Conclusion with CTA
  - [ ] Author bio
  - [ ] Related articles links
  - Comments: ____________

- [ ] **Internal and external linking**
  - [ ] 3-5 internal links included
  - [ ] External links to authoritative sources
  - [ ] Anchor text descriptive and relevant
  - Comments: ____________

- [ ] **Calls-to-action present**
  - [ ] Clear CTA in conclusion
  - [ ] CTA relevant to content
  - [ ] CTA actionable (links/forms functional)
  - Comments: ____________

### 6. Visual Assets 🖼️

- [ ] **Featured image meets specifications**
  - [ ] Dimensions: 1200x630px minimum
  - [ ] File size optimized (<200KB)
  - [ ] Alt text descriptive (100+ characters)
  - [ ] Licensed/owned (no copyright issues)
  - [ ] Visually appealing and on-brand
  - Comments: ____________

- [ ] **Images within post optimized**
  - [ ] All images have alt text
  - [ ] File sizes optimized
  - [ ] Dimensions appropriate
  - [ ] Captions provided where helpful
  - Comments: ____________

### 7. Technical & Formatting ⚙️

- [ ] **HTML/Markdown formatting correct**
  - [ ] Proper heading hierarchy
  - [ ] Bold/italics used appropriately
  - [ ] Lists formatted correctly
  - [ ] Code blocks (if any) properly formatted
  - Comments: ____________

- [ ] **Mobile responsive**
  - [ ] Tested on mobile device
  - [ ] Text readable on small screen
  - [ ] Images scale appropriately
  - [ ] No layout issues on mobile
  - Comments: ____________

- [ ] **Accessibility standards met**
  - [ ] Images have alt text
  - [ ] Color contrast sufficient
  - [ ] Links distinguishable from text
  - [ ] No content in images only
  - Comments: ____________
