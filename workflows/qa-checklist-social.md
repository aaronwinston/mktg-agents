## MAIN CHECKLIST

### 1. Copy Quality 📝

- [ ] **Grammar and spelling perfect**
  - [ ] No spelling errors
  - [ ] No grammar mistakes
  - [ ] Punctuation correct
  - [ ] Homophones used correctly (their/there/they're)
  - Comments: ____________

- [ ] **Tone appropriate for platform**
  - [ ] LinkedIn: Professional, insightful
  - [ ] Twitter: Conversational, timely
  - [ ] Instagram: Visual-focused, personable
  - [ ] TikTok: Authentic, trending-aware
  - Comments: ____________

- [ ] **Copy is clear and concise**
  - [ ] Message immediately clear
  - [ ] No jargon (unless explained)
  - [ ] Scannable (especially for LinkedIn/Twitter)
  - [ ] Call-to-action clear (if included)
  - Comments: ____________

- [ ] **Platform character limits observed**
  - [ ] LinkedIn: <3000 characters
  - [ ] Twitter: <280 characters (if single tweet)
  - [ ] Instagram: <2200 characters
  - [ ] TikTok: Caption clear (not primary text)
  - Comments: ____________

### 2. Visual Assets 🖼️

- [ ] **Images meet platform specifications**
  - [ ] Correct dimensions for platform
  - [ ] File size optimized (<1MB for social)
  - [ ] Format compatible (JPG, PNG)
  - [ ] Image quality high (no pixelation)
  - [ ] Text readable on small screen
  - Comments: ____________

- [ ] **Design is professional and branded**
  - [ ] Consistent with brand guidelines
  - [ ] Colors on-brand
  - [ ] Logo/watermark if appropriate
  - [ ] No copyright issues
  - Comments: ____________

- [ ] **Video meets specifications**
  - [ ] Platform: Video length appropriate
    - [ ] LinkedIn: <10 minutes
    - [ ] Twitter: <2:20
    - [ ] Instagram: <60 seconds (feed), <15 sec (Reel)
    - [ ] TikTok: <10 minutes
  - [ ] Resolution: 1080x1920 vertical recommended
  - [ ] Audio: Clear and at appropriate volume
  - [ ] Captions: Included (required for accessibility)
  - Comments: ____________

- [ ] **Accessibility standards met**
  - [ ] No text-only images (provide caption)
  - [ ] Color contrast sufficient
  - [ ] Videos have captions or subtitles
  - [ ] Alt text will be provided (for platforms that support)
  - Comments: ____________

### 3. Content Appropriateness ✓

- [ ] **No brand safety issues**
  - [ ] No controversial topics (unless intentional brand position)
  - [ ] No negative language about competitors
  - [ ] No misleading or false claims
  - [ ] Appropriate for all audience segments
  - Comments: ____________

- [ ] **Authenticity and accuracy**
  - [ ] Claims are truthful and substantiated
  - [ ] Quotes accurate (if used)
  - [ ] Statistics properly sourced
  - [ ] No exaggeration
  - Comments: ____________

- [ ] **Platform-specific appropriateness**
  - [ ] LinkedIn: Professional, substantive
  - [ ] Twitter: Engaging, not overly promotional
  - [ ] Instagram: Visually compelling, not overly salesy
  - [ ] TikTok: Authentic, not too corporate
  - Comments: ____________

### 4. Hashtag Strategy 🏷️

- [ ] **Hashtags researched and relevant**
  - [ ] At least 3 hashtags included
  - [ ] Hashtags currently trending (checked before posting)
  - [ ] No misspelled hashtags
  - [ ] Mix of broad and niche hashtags
  - Comments: ____________

- [ ] **Hashtag placement optimal**
  - [ ] LinkedIn: 3-5 hashtags (in text or first comment)
  - [ ] Twitter: 1-2 hashtags (max, integrated naturally)
  - [ ] Instagram: 10-30 hashtags (in first comment)
  - [ ] TikTok: 3-5 hashtags (in description)
  - Comments: ____________

- [ ] **Branded hashtag used (if applicable)**
  - [ ] Campaign-specific hashtag included
  - [ ] Hashtag consistent across platforms
  - Comments: ____________

### 5. Links & CTAs 🔗

- [ ] **All links functional**
  - [ ] URLs tested (no 404 errors)
  - [ ] Links point to correct destination
  - [ ] Links use proper tracking (UTM parameters)
  - [ ] Links load quickly
  - Comments: ____________

- [ ] **CTA is clear and actionable**
  - [ ] CTA tells user what to do ("Click here," "Learn more," "Register")
  - [ ] CTA relevant to content
  - [ ] Only one primary CTA per post
  - [ ] CTA button working (if applicable)
  - Comments: ____________

- [ ] **Link shortener (if used) configured**
  - [ ] Shortener tested
  - [ ] Link displays correctly
  - [ ] Analytics tracking enabled
  - Comments: ____________

### 6. Engagement Planning 💬

- [ ] **Response plan documented**
  - [ ] Who will monitor comments?
  - [ ] Response time SLA defined (e.g., within 24 hours)
  - [ ] Response templates prepared for common questions
  - [ ] Escalation path documented
  - Comments: ____________

- [ ] **Engagement hooks included (where appropriate)**
  - [ ] Question posed to encourage comments
  - [ ] Poll or survey included
  - [ ] Invitation to share opinion
  - Comments: ____________

### 7. Technical & Platform Setup ⚙️

- [ ] **Post scheduled correctly (if using scheduling tool)**
  - [ ] Correct date and time selected
  - [ ] Platform confirmed
  - [ ] Preview checked before scheduling
  - [ ] Timezone correct
  - Comments: ____________

- [ ] **Posting time optimized**
  - [ ] LinkedIn: Weekday 8am-1pm
  - [ ] Twitter: Morning or lunch time (varies by audience)
  - [ ] Instagram: Afternoon/evening
  - [ ] TikTok: Peak times (often 6-10pm)
  - Comments: ____________

- [ ] **Account/credentials verified**
  - [ ] Posting to correct account
  - [ ] Permissions verified
  - [ ] Approval workflow completed
  - Comments: ____________

### 8. Compliance & Brand Safety ⚖️

- [ ] **Platform ToS compliance**
  - [ ] No spam or repetitive posting
  - [ ] No engagement manipulation (follow/unfollow schemes)
  - [ ] No self-promotion only
  - [ ] Content type allowed on platform
  - Comments: ____________

- [ ] **Brand voice consistency**
  - [ ] Matches previous posts on this platform
  - [ ] Tone appropriate for brand
  - [ ] No controversial personal opinions
  - Comments: ____________

- [ ] **Legal requirements met**
  - [ ] FTC disclosure if sponsored/affiliate
  - [ ] No claims without substantiation
  - [ ] No promises of results
  - [ ] Privacy and cookie notices visible (if applicable)
  - Comments: ____________
