## MAIN CHECKLIST

### 1. Data Accuracy 📊

- [ ] **All data points verified**
  - [ ] Raw data source documents retained
  - [ ] Data entry accuracy verified (spot check: 100% of key numbers)
  - [ ] Calculations double-checked (formulas reviewed)
  - [ ] No typos in numbers
  - [ ] Decimal places correct
  - Comments: ____________

- [ ] **Sourcing is transparent and traceable**
  - [ ] Data source identified for every statistic
  - [ ] Source URLs provided or accessible
  - [ ] Publication date of source recorded
  - [ ] Source credibility documented
  - Comments: ____________

- [ ] **No data manipulation or misrepresentation**
  - [ ] Statistics presented in context
  - [ ] Cherry-picking avoided (full data set shown)
  - [ ] Statistical significance noted (if applicable)
  - [ ] Confidence intervals documented
  - Comments: ____________

### 2. Visualization Quality 📈

- [ ] **Charts and graphs accurately represent data**
  - [ ] Axes correctly labeled with units
  - [ ] Scale is honest (not artificially amplifying trends)
  - [ ] Data points plotted accurately
  - [ ] Legend clear and complete
  - [ ] Chart type appropriate for data (not misleading)
  - Comments: ____________

- [ ] **Visual design is professional**
  - [ ] Color scheme readable and professional
  - [ ] Color-blind accessible (not red-green only)
  - [ ] Fonts clear and sized appropriately
  - [ ] No clipart or unprofessional graphics
  - [ ] Branding consistent
  - Comments: ____________

- [ ] **All visualizations have supporting context**
  - [ ] Title clearly describes what data shows
  - [ ] Footnotes explain methodology (if needed)
  - [ ] Data source cited below each graphic
  - [ ] Key insight summarized in text
  - Comments: ____________

### 3. Analysis Integrity 🔍

- [ ] **Conclusions supported by data**
  - [ ] Conclusions directly stated in data
  - [ ] No leaps in logic
  - [ ] Alternative explanations considered
  - [ ] Limitations of data acknowledged
  - Comments: ____________

- [ ] **Recommendations are justified**
  - [ ] Recommendations follow from analysis
  - [ ] Not opinion dressed as data
  - [ ] Multiple options presented (not one-way)
  - [ ] Trade-offs acknowledged
  - Comments: ____________

- [ ] **Methodology is sound**
  - [ ] Sample size adequate (documented)
  - [ ] Bias sources identified and mitigated
  - [ ] Confidence levels stated
  - [ ] Limitations clearly acknowledged
  - Comments: ____________

### 4. Clarity & Communication 📝

- [ ] **Analysis is understandable to target audience**
  - [ ] Technical jargon explained
  - [ ] Assumptions stated
  - [ ] Key findings highlighted (not buried)
  - [ ] Summary accessible to non-experts
  - Comments: ____________

- [ ] **Structure is logical and clear**
  - [ ] Executive summary provides high-level overview
  - [ ] Findings organized by theme or importance
  - [ ] Transitions between sections smooth
  - [ ] Conclusion summarizes key points
  - Comments: ____________

- [ ] **Writing is error-free**
  - [ ] Grammar and spelling perfect
  - [ ] Consistent terminology throughout
  - [ ] Numbers spelled out or numerals (consistent)
  - [ ] No ambiguous pronouns or references
  - Comments: ____________

### 5. Context & Framing 🎯

- [ ] **Data presented in appropriate context**
  - [ ] Historical context (trends over time)
  - [ ] Comparative context (vs. competitors, industry)
  - [ ] Causal relationships distinguished from correlations
  - [ ] Impact quantified (not just "increased")
  - Comments: ____________

- [ ] **No misleading framing**
  - [ ] Headlines not sensationalized
  - [ ] Nuance preserved in simplification
  - [ ] Positive and negative findings both reported
  - [ ] Confidence levels transparent
  - Comments: ____________

- [ ] **Audience appropriate**
  - [ ] Language level matches audience expertise
  - [ ] Industry-specific context provided
  - [ ] Relevance to reader clearly stated
  - Comments: ____________

### 6. Methodology Documentation 📋

- [ ] **Research methodology clearly documented**
  - [ ] Research questions stated
  - [ ] Sample description (size, selection method)
  - [ ] Data collection method explained
  - [ ] Analysis approach described
  - [ ] Timeline of research documented
  - Comments: ____________

- [ ] **Limitations acknowledged**
  - [ ] Sample limitations noted
  - [ ] Timing limitations stated (data is X months old)
  - [ ] Geographic/scope limitations clear
  - [ ] Potential sources of bias identified
  - Comments: ____________

- [ ] **Reproducibility possible**
  - [ ] Enough detail for peer to understand approach
  - [ ] Methods would produce similar results if repeated
  - [ ] Data and findings are not contradictory
  - Comments: ____________

### 7. Sourcing & Attribution 📚

- [ ] **All sources properly credited**
  - [ ] Citations present for all data points
  - [ ] Citation format consistent and complete
  - [ ] Full publication details provided
  - [ ] Access information included (URLs, dates)
  - Comments: ____________

- [ ] **Distinguishing original from cited analysis**
  - [ ] Own analysis clearly marked as such
  - [ ] Third-party data clearly attributed
  - [ ] Quotations properly cited
  - [ ] Paraphrasing properly attributed
  - Comments: ____________

- [ ] **Bibliography/references complete**
  - [ ] All sources cited in text appear in references
  - [ ] All sources in references cited in text
  - [ ] Format consistent and complete
  - [ ] Sources credible and authoritative
  - Comments: ____________

### 8. Compliance & Ethics ⚖️

- [ ] **No ethical concerns**
  - [ ] No privacy violations (anonymization verified)
  - [ ] No conflicts of interest undisclosed
  - [ ] Data handling ethical
  - [ ] No selective reporting to support predetermined conclusion
  - Comments: ____________

- [ ] **Legal compliance**
  - [ ] No copyright violations
  - [ ] Data usage complies with terms of use
  - [ ] Any required permissions obtained
  - [ ] Confidential information protected
  - Comments: ____________

- [ ] **Claims substantiated**
  - [ ] Any performance claims backed by data
  - [ ] Causality claims justified
  - [ ] ROI/savings calculations transparent
  - [ ] Disclaimers present where needed
  - Comments: ____________

### 9. Comparison & Benchmarking 📊

- [ ] **Comparisons are fair and valid**
  - [ ] Like compared to like
  - [ ] Differences noted (if comparing different methodologies)
  - [ ] Time periods the same (or differences explained)
  - [ ] Units and metrics consistent
  - Comments: ____________

- [ ] **Benchmarks are appropriate**
  - [ ] Benchmarks from credible sources
  - [ ] Industry benchmarks compared to industry data
  - [ ] Outdated benchmarks flagged
  - [ ] Benchmark selection justified
  - Comments: ____________

### 10. Interactive Elements (if applicable) 🎯

- [ ] **Calculators/tools are accurate**
  - [ ] Formula tested with known values
  - [ ] Outputs verified against manual calculations
  - [ ] Edge cases handled appropriately
  - [ ] Rounding documented
  - Comments: ____________

- [ ] **Tools are usable**
  - [ ] Instructions clear
  - [ ] Inputs validated (no errors on bad input)
  - [ ] Output understandable
  - [ ] Mobile responsive
  - Comments: ____________
