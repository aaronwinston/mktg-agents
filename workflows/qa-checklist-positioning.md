## MAIN CHECKLIST

### 1. Strategic Soundness 🎯

- [ ] **Core strategic question is clear**
  - [ ] Question clearly defined in executive summary
  - [ ] Question is strategically relevant
  - [ ] Question is answerable with available data
  - [ ] Decision implications clear
  - Comments: ____________

- [ ] **Recommendations are strategically sound**
  - [ ] Follow logically from analysis
  - [ ] Address the core question
  - [ ] Realistic and implementable
  - [ ] Aligned with company capabilities
  - Comments: ____________

- [ ] **Strategic options are presented fairly**
  - [ ] 2-4 genuine alternatives presented
  - [ ] Trade-offs clearly articulated
  - [ ] No straw man options included
  - [ ] All options fairly analyzed (not biased)
  - Comments: ____________

- [ ] **Risk assessment is comprehensive**
  - [ ] Major risks identified
  - [ ] Likelihood and impact assessed
  - [ ] Mitigation strategies proposed
  - [ ] Unknown risks acknowledged
  - Comments: ____________

### 2. Evidence Quality 📊

- [ ] **Claims are substantiated**
  - [ ] Every major claim supported by evidence
  - [ ] 3+ sources for key findings
  - [ ] Primary data prioritized over secondary
  - [ ] Quality of evidence assessed (not all sources equal)
  - Comments: ____________

- [ ] **Research methodology is sound**
  - [ ] Research methods appropriate for question
  - [ ] Sample size adequate (if applicable)
  - [ ] Bias sources identified and mitigated
  - [ ] Research limitations acknowledged
  - Comments: ____________

- [ ] **Evidence is recent and relevant**
  - [ ] Data not more than 12-18 months old (unless historical)
  - [ ] Market context current
  - [ ] Competitive landscape reflects current state
  - [ ] Regulatory environment current
  - Comments: ____________

- [ ] **Sources are credible**
  - [ ] Primary sources prioritized
  - [ ] Secondary sources from reputable organizations
  - [ ] No self-interested sources used alone
  - [ ] Source expertise relevant to topic
  - Comments: ____________

### 3. Analysis Quality 🔍

- [ ] **Analysis is comprehensive**
  - [ ] Key factors considered
  - [ ] Internal and external perspectives included
  - [ ] Competitive landscape analyzed
  - [ ] Stakeholder viewpoints represented
  - Comments: ____________

- [ ] **Logic is sound**
  - [ ] Conclusions follow from premises
  - [ ] No logical fallacies present
  - [ ] Causal claims properly justified
  - [ ] Correlation/causation properly distinguished
  - Comments: ____________

- [ ] **Alternative perspectives considered**
  - [ ] Counterarguments addressed
  - [ ] Devil's advocate position considered
  - [ ] Different stakeholder views represented
  - [ ] Assumptions challenged, not assumed
  - Comments: ____________

- [ ] **Frameworks applied appropriately**
  - [ ] Strategic frameworks (SWOT, Porter's, etc.) properly used
  - [ ] Framework limitations acknowledged
  - [ ] Multiple frameworks used for triangulation
  - [ ] Framework insights synthesized clearly
  - Comments: ____________

### 4. Clarity & Communication 📝

- [ ] **Executive summary enables quick understanding**
  - [ ] Core question clearly stated
  - [ ] Key findings summarized (3-5 bullets)
  - [ ] Main recommendation stated
  - [ ] Timeline for decision/implementation
  - [ ] Can be understood standalone
  - Comments: ____________

- [ ] **Document is well-organized**
  - [ ] Clear logical flow
  - [ ] Sections build on each other
  - [ ] Key insights highlighted (not buried)
  - [ ] Appendices contain supporting detail
  - Comments: ____________

- [ ] **Writing is clear and professional**
  - [ ] Grammar and spelling perfect
  - [ ] Jargon either defined or avoided
  - [ ] Sentences are clear and concise
  - [ ] Tone is professional and objective
  - Comments: ____________

- [ ] **Visuals support understanding**
  - [ ] Charts and graphs accurately represent data
  - [ ] Visuals are labeled and sourced
  - [ ] Visuals break up text appropriately
  - [ ] Visual design is professional
  - Comments: ____________

### 5. Audience Appropriateness 👥

- [ ] **Content matches audience expertise**
  - [ ] Technical depth appropriate for audience
  - [ ] Context provided for non-expert readers
  - [ ] Assumptions about knowledge documented
  - [ ] Terminology consistent and defined
  - Comments: ____________

- [ ] **Relevance to audience clear**
  - [ ] Why this matters to reader explained
  - [ ] Business impact articulated
  - [ ] Implementation implications clear
  - [ ] Resource/budget implications identified
  - Comments: ____________

- [ ] **Actionability for decision-makers**
  - [ ] Recommendation clearly stated
  - [ ] Next steps defined
  - [ ] Decision required clearly articulated
  - [ ] Timeline for decision documented
  - Comments: ____________

### 6. Assumptions & Limitations 🔮

- [ ] **Key assumptions documented**
  - [ ] All major assumptions listed
  - [ ] Reasonableness of assumptions justified
  - [ ] Impact of assumption changes noted
  - [ ] Most critical assumptions highlighted
  - Comments: ____________

- [ ] **Limitations acknowledged**
  - [ ] Data limitations documented
  - [ ] Scope limitations clear
  - [ ] Time constraints noted
  - [ ] Information gaps acknowledged
  - Comments: ____________

- [ ] **Uncertainty properly communicated**
  - [ ] Confidence levels indicated
  - [ ] Scenarios with different outcomes presented
  - [ ] Monitoring metrics identified for risk areas
  - Comments: ____________

### 7. Internal Alignment ✓

- [ ] **Aligned with company strategy**
  - [ ] Recommendations fit strategic vision
  - [ ] Recommendations leverage core competencies
  - [ ] Recommendations don't contradict existing commitments
  - Comments: ____________

- [ ] **Feasibility assessed**
  - [ ] Resource requirements documented
  - [ ] Organizational capability assessed
  - [ ] Implementation timeline realistic
  - [ ] Success metrics defined
  - Comments: ____________

- [ ] **Stakeholder perspectives incorporated**
  - [ ] Key stakeholders interviewed (if applicable)
  - [ ] Concerns documented and addressed
  - [ ] Buy-in likely from key leaders
  - Comments: ____________

### 8. Competitive Analysis (if applicable) 🏆

- [ ] **Competitive landscape accurately assessed**
  - [ ] Key competitors identified
  - [ ] Competitive strengths/weaknesses fair
  - [ ] Competitive positioning clear
  - [ ] Market share/positioning data accurate
  - Comments: ____________

- [ ] **Differentiation strategy clear**
  - [ ] Competitive advantage articulated
  - [ ] Defensibility of advantage assessed
  - [ ] Barrier to competition identified
  - Comments: ____________

### 9. Compliance & Ethics ⚖️

- [ ] **No ethical concerns**
  - [ ] Analysis objective, not manipulated to support conclusion
  - [ ] Conflicts of interest disclosed
  - [ ] Proprietary information protected
  - Comments: ____________

- [ ] **Legal/regulatory compliance**
  - [ ] Relevant regulations considered
  - [ ] Compliance requirements documented
  - [ ] Any regulatory risks identified
  - Comments: ____________

- [ ] **Claims substantiated**
  - [ ] No unsupported assertions
  - [ ] Performance claims backed by evidence
  - [ ] Financial projections based on documented assumptions
  - Comments: ____________

### 10. Implementation Readiness 🚀

- [ ] **Implementation roadmap realistic**
  - [ ] Phasing logical and manageable
  - [ ] Dependencies identified
  - [ ] Critical path clear
  - [ ] Resource allocation realistic
  - Comments: ____________

- [ ] **Success metrics defined**
  - [ ] KPIs measurable and achievable
  - [ ] Baseline captured for comparison
  - [ ] Measurement methodology documented
  - [ ] Frequency of measurement defined
  - Comments: ____________

- [ ] **Governance/accountability clear**
  - [ ] Owner/sponsor identified
  - [ ] Decision authority clear
  - [ ] Escalation path documented
  - [ ] Review cadence defined
  - Comments: ____________
