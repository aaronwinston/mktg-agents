# Newsletter playbook

## Use this playbook for
- Regular newsletter issues
- Product update digests
- Curated content roundups

## Input

Use `briefs/newsletter-brief-template.md`.

## Skill chain

```
newsletter brief
→ managing-editor
→ content-repurposer
→ lifecycle-email-writer
→ copy-chief
→ final-publish-reviewer
```

## Steps

### 1. Editorial planning
Use `skills/editorial/managing-editor/SKILL.md`.

### 2. Content selection and repurposing
Use `skills/specialization/content-repurposer/SKILL.md`.

### 3. Email writing
Use `skills/specialization/lifecycle-email-writer/SKILL.md`.

### 4. Copy edit
Use `skills/editorial/copy-chief/SKILL.md`.

### 5. Final review
Use `skills/quality/final-publish-reviewer/SKILL.md`.

## Done means
- Issue structure is approved
- Subject line and preview text are finalized
- All links are verified
- Send is scheduled
