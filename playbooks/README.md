# Playbooks

Playbooks define the workflow for each major content type.

Each playbook includes:
- Use cases
- Required input
- Skill chain
- Steps
- Required rubrics
- Done definition

## Available playbooks

| Playbook | Use for |
|---|---|
| blog-production.md | Developer blogs, technical explainers, SEO content |
| product-launch.md | Product, feature, integration, partner launches |
| thought-leadership.md | Founder and executive POV content |
| case-study.md | Customer stories and case studies |
| analyst-response.md | Analyst briefings and inquiry responses |
| founder-social.md | Founder and executive social content |
| social-distribution.md | Repurposing assets for social channels |
| newsletter.md | Newsletter issues |
| campaign.md | Multi-channel campaigns |
| technical-guide.md | Technical how-to guides and tutorials |
