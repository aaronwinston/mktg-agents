# Social distribution playbook

## Use this playbook for
- Repurposing a published asset for social
- Building a distribution plan from a source asset

## Input

Source asset (blog, announcement, report).

## Skill chain

```
source asset
→ content-repurposer
→ social-editor
→ copy-chief
→ claims-risk-reviewer
→ content-ops-manager
```

## Steps

### 1. Repurpose
Use `skills/specialization/content-repurposer/SKILL.md`.

### 2. Social edit
Use `skills/specialization/social-editor/SKILL.md`.

### 3. Copy edit
Use `skills/editorial/copy-chief/SKILL.md`.

### 4. Claims review
Use `skills/quality/claims-risk-reviewer/SKILL.md`.

### 5. Operations
Use `skills/editorial/content-ops-manager/SKILL.md`.

## Done means
- All social variants are ready
- Claims in derivative content are verified
- Schedule and owner are defined
