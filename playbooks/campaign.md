# Campaign playbook

## Use this playbook for
- Multi-channel content campaigns
- Demand generation campaigns
- Product education campaigns

## Input

Use `briefs/campaign-brief-template.md`.

## Skill chain

```
campaign brief
→ editorial-director
→ ai-researcher
→ narrative-consistency-reviewer
→ dev-copywriter or launch-comms-writer
→ content-repurposer
→ social-editor
→ lifecycle-email-writer
→ claims-risk-reviewer
→ content-ops-manager
```

## Steps

### 1. Editorial direction
Use `skills/editorial/editorial-director/SKILL.md`.

### 2. Research
Use `skills/foundation/ai-researcher/SKILL.md`.

### 3. Narrative consistency
Use `skills/quality/narrative-consistency-reviewer/SKILL.md`.

### 4. Asset drafting
Use `skills/foundation/dev-copywriter/SKILL.md` or `skills/specialization/launch-comms-writer/SKILL.md`.

### 5. Repurpose across channels
Use `skills/specialization/content-repurposer/SKILL.md`.

### 6. Social
Use `skills/specialization/social-editor/SKILL.md`.

### 7. Email
Use `skills/specialization/lifecycle-email-writer/SKILL.md`.

### 8. Claims review
Use `skills/quality/claims-risk-reviewer/SKILL.md`.

### 9. Operations
Use `skills/editorial/content-ops-manager/SKILL.md`.

## Done means
- All campaign assets are drafted and reviewed
- Claims are cleared
- Distribution calendar is confirmed
- Owner for each channel is assigned
