# Founder social playbook

## Use this playbook for
- Founder LinkedIn posts
- Executive X threads
- Founder POV content from source material

## Input

Use `briefs/founder-social-brief-template.md`.

## Skill chain

```
source material
→ ai-researcher (if topic-driven)
→ founder-x-recap (if source is X activity)
→ executive-comms-writer
→ claims-risk-reviewer
→ copy-chief
→ social-editor
```

## Steps

### 1. Research (if topic-driven)
Use `skills/foundation/ai-researcher/SKILL.md`.

### 2. X recap (if source is X activity)
Use `skills/foundation/founder-x-recap/SKILL.md`.

### 3. Executive draft
Use `skills/specialization/executive-comms-writer/SKILL.md`.

### 4. Claims review
Use `skills/quality/claims-risk-reviewer/SKILL.md`.

### 5. Copy edit
Use `skills/editorial/copy-chief/SKILL.md`.

### 6. Social variants
Use `skills/specialization/social-editor/SKILL.md`.

## Required rubrics
- `rubrics/brand-fit.md`

## Done means
- Executive has approved the draft
- Claims are reviewed
- Post is ready to schedule
