# Thought leadership playbook

## Use this playbook for
- Founder or executive bylines
- Category-defining POV content
- Opinionated technical perspectives
- Trend analysis and market commentary

## Input

Source material: research, founder notes, product insight, market observation.

## Skill chain

```
idea or source material
→ editorial-director
→ ai-researcher
→ executive-comms-writer
→ narrative-consistency-reviewer
→ copy-chief
→ claims-risk-reviewer
→ social-editor
```

## Steps

### 1. Editorial direction
Use `skills/editorial/editorial-director/SKILL.md`.
Thought leadership requires a clear, defensible POV. Define it before drafting.

### 2. Research
Use `skills/foundation/ai-researcher/SKILL.md`.
Ground the POV in current evidence, trends, or data.

### 3. Executive draft
Use `skills/specialization/executive-comms-writer/SKILL.md`.
The draft must sound like the executive, not like a brand document.

### 4. Narrative consistency
Use `skills/quality/narrative-consistency-reviewer/SKILL.md`.

### 5. Copy edit
Use `skills/editorial/copy-chief/SKILL.md`.

### 6. Claims review
Use `skills/quality/claims-risk-reviewer/SKILL.md`.

### 7. Distribution
Use `skills/specialization/social-editor/SKILL.md`.

## Required rubrics
- `rubrics/editorial-quality.md`
- `rubrics/brand-fit.md`
- `rubrics/distribution-readiness.md`

## Done means
- Executive has reviewed and approved the draft
- POV is defensible and claims are sourced
- Distribution plan is ready
