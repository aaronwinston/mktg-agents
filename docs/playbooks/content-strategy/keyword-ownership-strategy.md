## KEYWORD DENSITY & PLACEMENT GUIDE

### Article Structure
**For each article targeting 1-3 keywords:**

**SEO Title (60 chars):**
- Include primary keyword
- Make compelling/clickable
- Example: "Agent Harness Engineering: Building Reliable AI Systems"

**Meta Description (160 chars):**
- Include primary + secondary keywords
- Clear value proposition
- Example: "Learn agent harness architecture patterns and design principles for multi-agent systems"

**H1 (Article title):**
- Include primary keyword

**H2 Headers (3-5 per article):**
- Weave in secondary keywords naturally
- Examples for "agent harness" article:
  - "Understanding Agent Harness Architecture"
  - "Tool Calling Within Your Harness"
  - "Observable Harness Engineering Patterns"

**Body (2K-5K words):**
- Natural keyword placement (0.8-1.2%)
- Intro: 1-2 mentions
- Section headers: Embedded
- Body: Distributed throughout
- Conclusion: 1-2 mentions

**Alt text on images:**
- Include keyword where relevant
- Example: "Agent harness architecture diagram showing orchestration layer"

### Keyword Co-occurrence
Strategically use related keywords together:
- "agent harness" + "orchestration layer"
- "multi-agent system design" + "memory architecture"
- "tool calling architecture" + "harness engineering"

## EXTERNAL LINKING STRATEGY

**Goal:** Build authority + relevance signals

### Link to:
- **3-5 authority sources** per article (industry leaders, research papers, frameworks)
- **Reference docs** (Python docs, AI frameworks, etc. where applicable)
- **Related tutorials** on other platforms (Dev.to, Medium, etc. - competitor intel)

### Prioritize:
1. Links to **research papers** (if technical topic) — credibility
2. Links to **framework documentation** (relevant to readers)
3. Links to **industry reports** (Gartner, analyst firms)
4. Links to **thought leaders** in the space (other Aparna-level voices)

### Avoid:
- Keyword stuffing in links ("click here for orchestration layer tips" — NO)
- Linking to direct competitors
- More than 10-15 outbound links per article
